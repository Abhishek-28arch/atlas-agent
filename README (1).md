## в—Ҳ What Is JARVIS?

**JARVIS** is a fully local, privacy-first AI automation system that runs entirely on your own hardware вҖ” no cloud, no subscriptions, no data leaks. Think Iron Man's assistant, but real, open-source, and running on a вӮ№0/month budget.

It combines a **local Large Language Model** (Qwen 3.5:4B via Ollama), **vector memory** (ChromaDB RAG), **computer automation** (PyAutoGUI), **messaging bots** (Telegram, Discord), and a **task planning engine** into one unified system that can:

- рҹҺӨ **Listen** to your voice and respond
- рҹ§  **Remember** everything you tell it вҖ” forever
- рҹ–ҘпёҸ **Control** your computer autonomously
- рҹ“ұ **Receive commands** from your phone via Telegram
- рҹ’ј **Automate** job applications, emails, GitHub, and more
- рҹ”Җ **Break down** complex goals into executable steps

```
You say:  "Apply to 10 Python developer jobs today"
JARVIS:   вҶ’ Searches LinkedIn/Indeed for matching jobs
          вҶ’ Reads each job description
          вҶ’ Customizes your resume for each role
          вҶ’ Writes a unique cover letter
          вҶ’ Logs everything to your spreadsheet
          вҶ’ Reports back: "Done. Applied to 10 jobs."
```

## в—Ҳ Quick Start

### Prerequisites

```bash
# Make sure you have:
# - Pop!_OS or any Linux distro
# - Python 3.11+
# - NVIDIA GPU (any VRAM)
# - Git installed

python3 --version    # Should be 3.11+
nvidia-smi           # Should show your GPU
```

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/jarvis.git
cd jarvis

# 2. Run the one-command setup
chmod +x setup.sh
./setup.sh

# 3. Configure your settings
cp config.example.yaml config.yaml
nano config.yaml   # Add your Telegram token, etc.

# 4. Add your documents to RAG
cp ~/Documents/my_resume.docx data/resume.docx
cp ~/Documents/notes.txt data/knowledge/

# 5. Start JARVIS
python main.py
```

### Setup Script (setup.sh)

```bash
#!/bin/bash
echo "вҡЎ Installing JARVIS..."

# Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Pull AI model
ollama pull qwen3.5:4b

# Python environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Install Playwright browsers
playwright install chromium

echo "вң… JARVIS is ready. Run: python main.py"
```

## в—Ҳ Safety System

JARVIS is powerful вҖ” so it has strict safety guardrails built in from day one.

```
SAFETY PRINCIPLES
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
1. Human approval required for all destructive actions
2. Blacklisted paths are never touched (SSH keys, passwords, /etc)
3. Every single action is logged with timestamp
4. Rate limiting prevents runaway loops
5. Kill switch stops everything instantly
6. Dry-run mode tests actions without executing
7. Only YOUR Telegram ID can send commands
```

```bash
# Emergency stop вҖ” kills all JARVIS processes instantly
python -m jarvis.safety.killswitch

# Or from Telegram:
/killswitch

# View action log
python -m jarvis.safety.logger --tail 50
```
